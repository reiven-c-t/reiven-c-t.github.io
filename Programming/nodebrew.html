Mac OSで任意のバージョンのNode.jsをインストール&管理するNodebrewというツールの使い方をまとめました。

環境情報

  • Mac OS 10.14 Mojave
  • Homebrew 1.7.6
  • nodebrew 1.0.1
  • Homebrewのインストールとアップグレードの方法
  • Node.jsによる文章校正ツール textlintの使い方

Nodebrewとは

NodebrewとはNode.jsのバージョンを管理するためのツールです。 最近の言語には開発サイクルは短いものが多いです。その中でもとくにNode.jsの開発サイクルは非常に短いです。Node.jsでのアプリケーション開発で、サードパーティライブラリを使用するとき、Node.jsのバージョンが違うことで挙動が違ったり、そもそも動かなかったりすることがよくあります。

そんな時、プログラムの実行環境をアプリケーションによって適宜切り替える必要が出てきます。これは他の言語でも言えることです。

Nodebrewのインストール方法

Mac OSにhomebrewが入っているものとします。

以下のコマンドでnodebrewをインストールできます。

brew install nodebrew;

もしnodebrewがすでにインストール済みの場合、次のコマンドでnodebrewをアップグレードできます。

brew upgrade nodebrew;

Nodebrewをインストールした後、まず、nodebrewのsetupのためのディレクトリを作成するコマンドを実行します。

/usr/local/opt/nodebrew/bin/nodebrew setup_dirs;

次にnodebrewへパスを通すため、ホームフォルダの.bash_profileファイルに次の行を追加します。

export PATH=$HOME/.nodebrew/current/bin:$PATH
export PATH=/usr/local/var/nodebrew/current/bin:$PATH

NodebrewでNode.jsをインストールする

まずNodebrewでインストールできるNode.jsの一覧をnodebrew ls-remoteで確認します。

$ nodebrew ls-remote
v0.0.1    v0.0.2    v0.0.3    v0.0.4    v0.0.5    v0.0.6
~~省略~~
v10.0.0   v10.1.0   v10.2.0   v10.2.1   v10.3.0   v10.4.0   v10.4.1   v10.5.0
v10.6.0   v10.7.0   v10.8.0   v10.9.0   v10.10.0  v10.11.0

io@v1.0.0 io@v1.0.1 io@v1.0.2 io@v1.0.3 io@v1.0.4 io@v1.1.0 io@v1.2.0 io@v1.3.0
io@v2.3.2 io@v2.3.3 io@v2.3.4 io@v2.4.0 io@v2.5.0
~~省略~~
io@v3.0.0 io@v3.1.0 io@v3.2.0 io@v3.3.0 io@v3.3.1

上の結果から、実行時点で最新のNode.js v10.11.0をインストールします。

nodebrew install v10.11.0

インストールが完了したら、nodebrewでローカルにNode.js v10.11.0が存在することを確認します。

nodebrew ls;
v6.12.0
v10.11.0

current: v6.12.0

上記のようにv6.12.0とv10.11.0がローカルに存在していることを確認しました。(v6.12.0は以前インストールしています。)

次にnodebrewで新たにインストールしたv10.11.0に切り替えます。

nodebrew use v10.11.0;

これでnode v10.11.0がセットされました。

現在のNode.jsのバージョンを確認してv10.11.0であれば成功です。

node --version;

また、同様に別バージョンのnodeをインストールし、nodebrew useコマンドで切り替えることもできます。またnodeの最新版をインストールするさい、直接バージョンを指定する以外にも、nodebrew install latestという方法もあります。

reiven@Reiven-Mac Downloads$ nodebrew ls
v6.12.0
v10.11.0

current: v10.11.0
reiven@Reiven-Mac Downloads$ nodebrew use v6.12.0
use v6.12.0
reiven@Reiven-Mac Downloads$ node --version
v6.12.0

ここまでのコマンドまとめ

# nodebrewインストール
brew install nodebrew;
/usr/local/opt/nodebrew/bin/nodebrew setup_dirs;
# path追加
echo 'export PATH=$HOME/.nodebrew/current/bin:$PATH' >> ~/.bash_profile;
echo 'export PATH=/usr/local/var/nodebrew/current/bin:$PATH' >> ~/.bash_profile;
# nodeのバージョン一覧確認
nodebrew ls-remote;
# nodeの最新版インストール&アクティベート
nodebrew install latest;
nodebrew use latest;
node --version;
