  • Mac OSにHomebrewをインストール&アップデートする方法

Mac OSでデファクトスタンダードと言っても過言ではない、パッケージ管理ツールであるHomebrewのインストール&アップデート方法をまとめて解説します。

  • 環境情報
  • Homebrewをインストール
  • Homebrew本体とHomebrewでインストールしたパッケージの一括アップデート

環境情報

  • Mac OS Mojave
  • ruby(Mac OSに標準インストール済み)

Homebrewをインストール

Homebrew
The missing package manager for macOS (or Linux).

上のサイトのスクリプトをコピペするだけです。

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Homebrew本体とHomebrewでインストールしたパッケージの一括アップデート

Homebrew本体とHomebrewでインストールしたパッケージで最新版が存在する物を一括でアップデートするbrew upgrade --allがあります。

brewコマンドを使用しているとたまーに、brewが古くなったとかコメントを吐いてくるので適宜upgradeすると良いでしょう。
